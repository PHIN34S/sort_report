Bubble Sort

Learn how elements "bubble up" to their correct positions through repeated swaps.

Bubble Sort Interactive

The Logic (C Implementation)

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                // Swap values
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}
