一、操作原理

氣泡排序法（Bubble Sort）是最直覺的比較排序演算法之一。
核心概念：反覆掃描陣列，每次比較相鄰兩個元素，若左邊 > 右邊則互換， 如此較大的元素就像泡泡一樣「浮」到陣列末端。

每完成一輪（Pass）掃描，當輪最大值就會確定就位於末端。 若某輪掃描中沒有發生任何交換，代表陣列已完全排序，可提前結束（最佳化版本）。

二、演算法偽代碼

function bubbleSort(arr):
  1n = arr.length
  2for i = 0 to n-2: // 共 n-1 輪
  3  swapped = false
  4  for j = 0 to n-2-i: // 已就位的不再比較
  5    if arr[j] > arr[j+1]:
  6      swap(arr[j], arr[j+1])
  7      swapped = true
  8  if not swapped: break // 提前結束最佳化

三、時間 & 空間複雜度

情況 時間複雜度 說明
最佳情況 (Best) O(n) 陣列已排序，第一輪無交換即結束
平均情況 (Average) O(n²) 元素隨機分布
最壞情況 (Worst) O(n²) 陣列完全逆序
空間複雜度 O(1) 原地排序，僅需常數額外空間
穩定性 穩定 ✓ 相等元素不改變原始相對順序
